# 2. 자료형

``` python
print(type("안녕하세요"))
print(type(100))
print(type(3.14))
print(type(True))
```

  자료형   예시
  -------- ----------
  str      "Python"
  int      10
  float    3.14
  bool     True

# 3. 연산자

``` python
print(a**b)
```

# 5. 조건문

``` python
year = int(input("연도 입력 : "))

if year % 4 == 0 and year % 100 != 0 or year % 400 == 0:
    print("윤년")
else:
    print("평년")
```

------------------------------------------------------------------------

# 6. 반복문

## for

``` python
for i in range(5):
    print(i)
```

## while

``` python
count = 1
while count <= 5:
    print(count)
    count += 1
```

# 14. 클래스

``` python

    def __init__(self, name):
        self.name = name

```

# VS Code에서 실행하기

1.  저장 (Ctrl + S)
2.  실행 (Ctrl + F5)
3.  결과는 터미널에서 확인

> **중요**: 파일(.py)에서는 `type()`처럼 반환만 하는 함수는 자동
> 출력되지 않습니다.

``` python
print(type("안녕하세요"))
```

처럼 `print()`를 사용해야 화면에 출력됩니다.
